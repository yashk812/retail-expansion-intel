# Retail Expansion Intel

Competitive intelligence dashboard for Baazar Kolkata — tracks 2,600+ stores across 8 retail chains.

## Pages
1. **Store Map** — Interactive map with clustering, filters by state/district/city, toggle companies
2. **Stats by Company** — Heatmap + bar charts comparing store counts across geographies  
3. **Expansion Insights** — Opportunity Score ranking of districts where BK should expand

## Running locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deployment
Deployed on [Streamlit Community Cloud](https://streamlit.io/cloud).
